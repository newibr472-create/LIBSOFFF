8. PATCH_LIB FORMAT (How to make hooks - IMPORTANT)
================================================================================
Tumhara format:
  PATCH_LIB("library_name.so", "0xOFFSET", "HEX BYTES");

Example:
  PATCH_LIB("libUE4.so", "0xBC2C054", "1F 20 03 D5");
  
  Library: libUE4.so, libanogs.so (jo bhi target ho)
  Offset:  Function ka VA address (hex, with 0x prefix)
  Hex:     Original bytes ko overwrite karne ke liye new bytes

Common patches:
  NOP:  1F 20 03 D5  (instruction ko skip karo)
  RET:  C0 03 5F D6  (function se turant return)
  MOV:  varies        (register value change)

How to find offsets:
  Method 1: ADRP xref scanning (strings dhundho → xref resolve → function entry)
  Method 2: PLT trace (import dhundho → PLT stub → BL scan in .text)
  Method 3: Ghidra/IDA decompilation (Codespace mein Ghidra chalao)
